# 💰 Financeiro • Terceirão

Sistema web para organização financeira do terceiro ano, criado para controlar o dinheiro arrecadado pela turma de forma simples, colaborativa e organizada.

## 📌 Sobre o projeto

O Financeiro • Terceirão permite que a turma acompanhe o dinheiro disponível, separando valores em diferentes categorias e mantendo um controle coletivo das finanças.

O sistema foi pensado para facilitar a organização de eventos, vendas, rifas, barracas e outros projetos da turma.

## ✨ Funcionalidades

### 🐷 Porquinhos

Os porquinhos funcionam como caixas separadas de dinheiro.

Exemplos:

- Barraca
- Rifa
- Formatura
- Reserva

Cada porquinho possui:

- Nome
- Descrição
- Valor em PIX
- Valor em dinheiro físico
- Observações

O sistema calcula automaticamente o valor total de cada porquinho.

---

### 💳 Caixa Geral

O Caixa Geral mostra:

- Total em PIX
- Total em dinheiro físico
- Valor total disponível

Os valores são calculados automaticamente com base nos porquinhos cadastrados.

---

### 📝 Pendências

Área para organizar informações importantes, como:

- Valores que ainda precisam ser recebidos
- Contas a pagar
- Observações financeiras

Permite:

- Criar pendências
- Editar
- Excluir
